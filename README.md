# Guizhong

A Discord bot that plays music from a queue.


## Getting started

Run bot:

```
uv run main.py
```

See [setup instructions](./SETUP.md) for more detailed instructions.

Testing code:

```
uv run pytest -v -s --cov=src --cov-branch --cov-report=xml --cov-report=html --cov-report=term
```

Formatting code:

```
black .
```
